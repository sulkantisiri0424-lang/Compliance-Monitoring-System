# Compliance Monitoring System

## Project Overview

The Compliance Monitoring System is a Multi-Agent AI-based application designed to automate compliance monitoring for financial institutions.

## Features

- Multi-Agent Architecture
- Transaction Monitoring Agent
- Communication Scanner (NLP)
- Regulatory Update Tracker
- Report Generator
- Dashboard & Analytics
- Audit Logging

## Technologies

- Python
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript
